How to migrate to TinTin++

Migrating your legacy scripts is easier than ever. Follow these simple steps to get started.

1

Select Source

Choose your legacy client (Powwow, PowTTY, or JMC) from the dropdown menu.

2

Paste Script

Paste your legacy code into the source area. Try our examples if you're unsure.

3

Copy & Run

Copy the modern TinTin++ output and paste it into your TinTin++ configuration file.

Why migrate?

TinTin++ is a powerful, modern MUD client that is actively maintained and supports advanced features like automapping, PCRE, and high-performance scripting.

  • ✔ High-performance parsing engine
  • ✔ Modern cross-platform compatibility
  • ✔ Advanced automapping and 24-bit color
  • ✔ Active community and ongoing development

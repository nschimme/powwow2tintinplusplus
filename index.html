Project Status

The converter has been upgraded with a robust tokenizer. It now supports complex nested structures, multi-line aliases, and real-world script patterns from MUME.

Supported Features

  • Recursive command parsing
  • Nested braces & parentheses
  • Numbered & named variables
  • Delayed substitutions (\$, \&)
  • Inline attribute mapping (attr)
  • Multiple command separators

Technical Details

  • Modern ESM architecture
  • Automated test suite (Vitest)
  • Benchmark validated (ElvenRunes)
  • Tailwind CSS UI
  • GitHub Actions deployment
